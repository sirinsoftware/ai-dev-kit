How to use. Scaffold a throwaway project with both agents, e.g. ./setup.sh ~/tmp/adk-qa --agents=codex,copilot --with-all-extras -y, fill in the {{…}} placeholders in its AGENTS.md, then work top-to-bottom. Set each test to Pass / Fail / Blocked / N A and jot notes — everything persists locally and exports to JSON or Markdown.
